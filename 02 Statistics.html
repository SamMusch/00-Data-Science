                                                          Ch 1: Intro

                                                          • Population: any specific collection of objects of interest
                                                          • Sample: any subset or subcollection of the population

                                                          Measurement: is a number or attribute computed for each member of a population or of a sample. The measurements of sample elements are collectively called the sample data.

                                                          • Parameter: is a number that summarizes some aspect of the population
                                                          • Statistic: a number computed from the sample data.

                                                          Ch 3: Probability

                                                          Random experiment: a mechanism that produces a definite outcome that cannot be predicted with certainty

                                                          • Sample space: set of all possible outcomes
                                                          • Event: subset of the sample space

                                                           

                                                          • Probability of an outcome: value between 0 and 1
                                                          • Probability of an event: sum of these values

                                                          Example

                                                           

                                                          Complement

                                                          Complement = 1 - Probability of an event

                                                          Intersection = Both A & B

                                                          Union = Either A or B

                                                          • P(A∪B)=P(A)+P(B)−P(A∩B)

                                                           

                                                          Conditional

                                                          Conditional: P(A|B)=P(A⋂B)P(B)

                                                          Independent when: P(A∩B) = P(A) · P(B)

                                                          Bernoulli (Just 1)

                                                          Curley Image

                                                          • Sample Space: X = Success or Failure {0, 1}
                                                          • p(Success) = p
                                                          • p(Failure) = q = 1 - p

                                                          Parameters

                                                          • Mean = p
                                                          • Variance = p * q

                                                          Image of equation

                                                          Suppose a random experiment has the following characteristics:

                                                          1. N: iid trials of a common procedure
                                                          2. 2 outcomes possible, one termed “success” and the other “failure”
                                                          3. p: Probability of success

                                                          Binomial random variable: number of successes in the trials

                                                          We can use the same probability as before, but we can also use the following simplified ones for binomials. (q = 1 - p)

                                                          • Mean = n * p
                                                          • Variance = n * p * q

                                                           

                                                          Proportions

                                                          Curley Image

                                                          Same thing, just divided by the sample size

                                                          • Mean = p
                                                          • Variance = p∗qn

                                                           

                                                          Poisson (Count in time)

                                                          Curley Image

                                                          Useful for count data that occurs infrequently. Looking for the number of successes in a given interval of time or space.

                                                          λ: Number of expected successes within the time interval

                                                          Parameters:

                                                          • Mean: λ
                                                          • Variance: λ

                                                           

                                                          Negative Binomial

                                                          Curley Image

                                                          How many trials until we get success?

                                                          r: Number of trials until we get our desired "Successes"

                                                          Parameters:

                                                          • Mean: rp
                                                          • Variance: r(1−p)p2
